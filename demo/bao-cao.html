Câu hỏi 5 · Khó khăn & cách xử lý

Debug thật — không đoán mò L3 ⭐

  • Metafield scope: write_metafields bị bỏ → hiểu merchant-owned cần scope, app-owned $app thì không → đổi reserved namespace.
  • 3 bug live (reproduce → root cause → fix → regression test): query bị # comment hết dòng · document.currentScript null trong module · theme-check literal <script.
  • Chọn đúng tool/skill: design-prototype khóa Polaris → tự author; codegraph overkill app nhỏ → dùng chọn lọc.
Takeaway: mỗi bug = reproduce + root cause + (nơi được) regression test. Judgment: biết khi nào KHÔNG dùng tool nặng.
